About MTR Topology Visualizer

Overview

MTR Topology Visualizer is a powerful tool for visualizing network topology and monitoring network performance. It combines the functionality of traceroute and ping, creating a comprehensive view of your network's structure and health.

The tool uses MTR (My TraceRoute) to trace the path between your server and various network endpoints, collecting data about each hop along the way. This data is then visualized in an interactive graph, allowing you to quickly identify bottlenecks, latency issues, and packet loss.

Features

  • Interactive Topology Graph: Visualize your network topology with an intuitive, interactive D3.js graph. Nodes represent network devices, and links show connections between them.
  • Real-time Monitoring: Monitor network performance in real-time with auto-refresh capabilities that update the topology graph at configurable intervals.
  • Agent Management: Easily add, remove, enable, or disable monitoring agents through the web interface.
  • Telegraf Integration: Automatically discover network endpoints from your Telegraf configuration files.
  • Filtering and Grouping: Filter the topology view by group or agent to focus on specific parts of your network.
  • Performance Metrics: View detailed performance metrics including latency and packet loss for each network segment.
  • Dashboard: Access a comprehensive dashboard with historical performance data and network statistics.

How It Works

MTR Topology Visualizer works by sending ICMP packets to target destinations with incrementing TTL (Time To Live) values. This allows the tool to map the entire path from source to destination, recording information about each router along the way.

The application consists of three main components:

  1. Core MTR Engine: Handles the sending and receiving of ICMP packets, calculating statistics, and building topology data.
  2. Storage Layer: Persists configuration and topology data in an SQLite database, allowing for historical analysis.
  3. Web Interface: Provides an intuitive user interface for visualizing topology data and managing agents.

System Requirements

  • Linux-based operating system
  • Python 3.6 or higher
  • Root privileges (required for raw socket operations)
  • Modern web browser (Chrome, Firefox, Safari, Edge)
